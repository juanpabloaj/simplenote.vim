simplenote.vim

A vim plugin for interacting with the Simplenote web service.

Screenshot

Install

Copy the simplenote.vim file into your plugin folder or create a bundle/simplenote.vim folder if you use pathogen.

Usage

For a detailed README see here

List notes:
:Simplenote -l

Update current note:
:Simplenote -u

Trash current note:
:Simplenote -d

Delete current note:
:Simplenote -D

Create new note from current buffer:
:Simplenote -n

Edit note tags:
:Simplenote -t
